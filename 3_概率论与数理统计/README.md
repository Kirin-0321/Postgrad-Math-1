## 学习建议

1. 3_概率论与数理统计是考研数学的重要内容，重点掌握概率计算和分布函数
2. 特别关注常见分布（正态分布、指数分布、泊松分布等）及其性质
3. 熟悉各种参数的计算方法
4. 理解大数定律与中心极限定理的应用
5. 掌握基本的统计推断方法
6. 重视概念理解，通过例题加深理解
7. 数理统计部分注重应用，掌握常用的估计与检验方法
