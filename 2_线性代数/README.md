## 学习建议

1. 2_线性代数是考研数学的重要组成部分，重点掌握矩阵运算和特征值
2. 注重概念和计算方法的结合
3. 掌握解题技巧和常用结论
4. 特别关注矩阵的相似对角化和二次型的标准形
5. 学习时注意建立几何直观，理解抽象概念的几何意义
6. 线性方程组、特征值问题和二次型问题是核心内容，要重点掌握
